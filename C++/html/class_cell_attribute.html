Detailed Description

細胞（セル）の属性情報.

Note:
細胞（セル）が持っている属性.（has-aの関係）.
MatrixCellからExpressionCellに情報を受け渡すのにも使用する.
受け渡す情報は以下.
・座標位置 横方向.
・座標位置 縦方向.
・セルの状態.
